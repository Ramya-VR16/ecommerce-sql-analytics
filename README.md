# E-commerce SQL Analytics

## Overview

This project analyzes e-commerce order, customer, and revenue data using SQL on Google BigQuery public datasets.

The objective is to demonstrate practical SQL skills used in Data Analyst roles by answering real business questions.

## Dataset

Source: Google BigQuery Public Dataset  
Table used: bigquery-public-data.thelook_ecommerce.orders

## Tools

SQL  
Google BigQuery  
GitHub

## Analysis Performed

Data quality validation  
Monthly order trends  
Month-over-month growth analysis  
New vs returning customer analysis  
Customer retention logic  
Revenue metrics including AOV  

## Project Structure

ecommerce-sql-analytics  
README.md  
sql folder containing all SQL queries  

## Key Skills Demonstrated

SQL data analysis  
Time series analysis  
Customer behavior analysis  
Business metrics and KPIs  

## How to Run

Run the SQL queries in Google BigQuery using the public dataset mentioned above.

## Purpose

This project was created as hands-on practice for Data Analyst roles.
