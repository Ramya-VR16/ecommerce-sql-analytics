# E-commerce SQL Analytics (BigQuery)

## Overview
This project analyzes e-commerce order, customer, and revenue data using SQL on Google BigQuery public datasets.

## Dataset
- bigquery-public-data.thelook_ecommerce.orders

## Key Analyses
- Data quality validation
- Monthly order trends
- Month-over-Month growth
- New vs returning customer analysis
- Customer retention
- Revenue metrics (AOV, revenue trends)

## Tools Used
- SQL (Google BigQuery)
- GitHub

## Key Insights
- Order volume and revenue show long-term growth
- Returning customers contribute increasing revenue share
- Retention patterns reveal customer behavior trends
